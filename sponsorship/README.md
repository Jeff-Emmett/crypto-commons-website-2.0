# CCA 2026 Sponsorship Materials

This folder contains the sponsorship pitch package for the Crypto Commons Gathering 2026.

## Files

| File | Purpose |
|------|---------|
| `CCA-2026-Sponsorship-Package.md` | Full sponsorship deck (Markdown) |
| `CCA-2026-Sponsorship-Package.html` | Styled HTML version |
| `CCA-2026-Sponsorship-Package.pdf` | Print-ready PDF |
| `CCA-2026-Sponsorship-OnePager.md` | Quick reference one-pager |
| `CCA-2026-Slides-Content.md` | Google Slides deck content |
| `outreach-email-template.md` | Email templates for outreach |
| `prospect-list.md` | Target sponsor list with priorities |

## How to Use

### For Outreach
1. Use `CCA-2026-Sponsorship-OnePager.md` for initial emails
2. Customize templates from `outreach-email-template.md`
3. Send full PDF package when requested
4. Reference `prospect-list.md` for targeting

### For Presentations
1. Open `CCA-2026-Slides-Content.md`
2. Create new Google Slides presentation
3. Copy slide-by-slide content
4. Apply suggested styling and design notes

### For Print
- Use `CCA-2026-Sponsorship-Package.pdf` for print or email attachment

## Sponsorship Tiers

| Tier | Investment | Key Benefits |
|------|------------|--------------|
| Steward Partner | €25,000+ | Keynote, workshop, 10 passes |
| Commons Builder | €15,000 | Panel, session, 6 passes |
| Regenerative Supporter | €7,500 | Lightning talk, 4 passes |
| Community Contributor | €3,000 | Website listing, 2 passes |
